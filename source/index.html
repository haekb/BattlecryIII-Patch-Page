Widescreen 1.04 Beta Patch

This patch is a combination of effort done by myself, along with help from the team over at TPC. Done in my spare time, it's been in development for quite a few months and is now finally at the point of a public beta release.

KEY CHANGES

  • Enhancements to the game's rendering code.
  • Smoother Gameplay
  • You can now use overlays like Steams or Discords.
  • Windowed mode!
  • Launch directly to editor with the EDITOR argument.

The entire changelog be found over here

INSTALLATION

This patch requires a fresh install of either the GOG or Steam version of Warlords Battlecry III. It is not compatible with any other patches.

Just drag the files into your Warlords Battlecry III directory.

SUPPORT

Please note this is an unofficial patch, it's NOT developed by Infinite Interactive or Enlight Interactive.

I'm providing limited support for bugs on the patch's moddb forums.

DOWNLOAD

You can find the download link at the moddb or at itch.io.
